# 看房记录

优先条件：

```md
1. 满 5
2. 朝南、南北通透
3. 10 年内
    - 这个条件筛了一大堆出去
4. 有电梯
5. 商品房
6. 850w 内
```

## 金桥

金桥地段还不错，但再往上跟高行的新房比又划不来。

#### 碧云新天地

```md
1. --- 780w
    - 从 900 降价到 780，6 次。看来房主急着出手
    - 最终 780 是因为楼上有个同户型的之前卖了 780。
2. 小区地面停车比较多，难出来。
3. 物业费 2.5
4. 800 米地铁，旁边有个翡翠坊，距离旁边金桥国际大概 1.5 公里。
5. 一梯两户，满五唯一。客厅有老人去世，23 年。
6. 小区另有一户 835w 的有抵押，正在解决。
```

#### 外外滩 - 碧玉澜庭

```md
1. 730w 左右
2. 15 年小区，带产权地下车位，电梯入库
3. 3 栋楼，9 个楼号。总高 18 层，8 楼
4. 要有电梯卡才行，电梯入户，入库。管理比较好
5. 两梯两户，跟邻居对门，公共面积有，但是不好占用
6. 南北通透，两房
7. 小区环境，楼间距都挺好。物业费 2.8
8. 后面金滩规划，现在正在搬，有噪音
    - 造船厂，后面建设休闲+写字楼类似的东西
9. 总结：
    - 算是比较满意的房子，小区环境好管理好，地段好，还送产权车位
    - 缺点就是房子户型不太好，面积只有 93 有点小。
```

## 御桥

-   御桥地段在中外环间，这里中外环靠的比较近。紧紧夹着。
-   仁和都市花园，想 100 平要 800w+了。

#### 颐盛御中环

```md
1. 小高层，人车分流。
2. 两梯两户
3. 后面施工工地，说是会建公园
4. 物业费 3.6
5. 门口有自己打的柜子
6. 车位 30w
7. 室内空间大，视野开阔，附近商区有一些，装修也奢华。会把 80%的家具留下来。
8. 看的是 999w 的房子，装修十分奢华。小区另有房子 800+的。
```

#### 绿洲康城亲水湾

```md
1. 带学区，小学中学，有会所健身房
2. 一梯两户，边套。户型方正，南北通透，各房间采光不错。
3. 车库 34w，月租 200，限高 2 米 2
4. 配套设施也都有，小区建成 7、8 年，
5. 门口都是小区，比较安静，但是出门去商区要走一点
6. 缺点是离外环高架近
    - 不是小区最边上的，从高架方向数在第二层
    - 客厅开窗能听到声音，卧室关窗隐约有声音
```

#### 阳光城愉景公馆

```md
1. 一梯两户，门口能放鞋架
2. 118 平，没看出来，房型不太好
3. 进门地方空间太浪费了
4. 小区进出门不方便，贴着马路就一栋
5. 总结：只有靠近马路的门面好看，其它的都不太行
```

## 北蔡

-   位于浦东中环-外环间，距离前滩、张江、陆家嘴均在 20 分钟车程内。但房价仅为前滩的 1/2。
-   山姆会员店（北蔡店，盒马 X 会员店、复地活力城等，三甲级上海第一妇婴保健院（东院）在 3 公里范围内
-   北蔡动迁房比较多，像是大型城中村集合地。
-   为数不多的优质小区是 陆家嘴锦绣里。

#### 陆家嘴锦绣里

```md
1. 小区内有商品房，动迁房
2. 地铁旁边，1 公里有个商区，三城里，3 公里巴黎春天。2 公里山姆。
3. 16 年小区，刚换的万科物业，物业费 2.58
4. 停车月租 200，地下 500，买的话 20 多 w
5. 朝西，两梯四户，动迁房
6. 小区环境不错，有篮球场
7. 小区 12 栋楼，围着中心花园
8. 小区共有 3 套房出售
    - 动迁房，光线非常好。但是动迁房！
    - 剩下 2 个商品房，边套两梯三户。都是朝东光线不行。
        - 据中介说动迁房/商品房区别，只是商品房门面有瓷砖。
        - 顶楼：顶楼 18，层高高 10-15 公分。顶楼这个厨房，卫生间一点光没有
        - 临街，阳台，厨房能听到车辆开过的声音，目前是开放商送的单层玻璃。
```

#### 大华锦绣华城

```md
1. 配套比较全，卖东西的比较多
2. 10 年的房子，
3. 动迁房，门口就有商区
4. 内部停车比较乱
5. 临高架，比较吵。两侧都能听到声音，临河，河水黄。
6. 一梯两户，比较老电梯，哒哒哒的。
7. 总结：唯一优势是户型比较方正
```
